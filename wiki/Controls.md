```text
    +-----------------------------------------+
    |              CONTROLS                   |
    +-----------------------------------------+
```

## Navigation

```text
+---------------+----------------------------------+
|    KEY        |    ACTION                        |
+---------------+----------------------------------+
|  Arrow Keys   |    Navigate menus                |
|  [TAB]        |    Open main menu                |
|  [ENTER]      |    Confirm selection             |
|  [ESC]        |    Go back / Cancel              |
+---------------+----------------------------------+
```
